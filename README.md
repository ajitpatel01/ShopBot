# ShopBot
WhatsApp AI Assistant for Local Businesses in India

## Stack
- Backend: Node.js + Express (localhost dev, Railway production)
- WhatsApp: whatsapp-web.js
- AI: Groq API — Llama 3.3 70B
- Database: Supabase (PostgreSQL + Auth + Realtime)
- Dashboard: Next.js 14 + Tailwind CSS (Vercel)
- Payments: Razorpay

## Setup
1. Clone this repo
2. cd backend && npm install
3. Copy .env.example to .env and fill in all values
4. Run schema.sql in your Supabase SQL editor
5. npm run dev
